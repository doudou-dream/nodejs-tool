## 1. 模板生成工具类(template-generate)

```sh
cd template-generate
pnpm init
pnpm add -D readline
node  run.js
```

## 2. 打包上传服务器(package-upload)

```sh
cd template-generate
pnpm init
pnpm add -D chalk scp2 ssh2 compressing ora fs path
node  run.js
```

## 2. 下载文件，解压、压缩文件(download-zip)

```sh
cd template-generate
pnpm init
pnpm add -D http https compressing
```
